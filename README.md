FinSense

Simple personal finance tracker built with React Native + TypeScript.

📌 Features
	•	Add and manage expenses
	•	Simple UI for tracking spending
	•	Basic navigation and screen structure

🚀 Getting Started

git clone https://github.com/PR-BHANU/FinSense.git
cd FinSense
npm install
npm start
npm run android   # or
npm run ios
